Planet - Vacation Planner

Planet - Vacation Planner was my first big project I completed at Devmountain. The demo for Planet is hosted using Amazon Web Services at Planet - Vacation Planner. The project is also available publicly on Github. It was built using JavaScript for both the front end and the back end by utilizing the back end framework Express.

Planet - Vacation Planner was built as a simple way to keep note on trip planning. It's process is simple. You set a budget and then you add plans to your list. The site automatically keeps track of your budget. The details section is then flexable enough for you to keep track of any additional information you might need.
